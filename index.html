Supported Hardware

Board:


MaTouch ESP32-S3 AMOLED with Touch 1.8” FT3168


Chip Family: ESP32-S3

Installation Steps

  • Connect board via USB.
  • Open Chrome or Edge.
  • Click Flash Firmware.
  • Select serial port.
  • Install firmware.
  • Wait until flashing completes.
  • Device will reboot automatically.

Browser Requirements

  • Google Chrome
  • Microsoft Edge
  • Opera
  • Brave
  • Web Serial enabled browser

Important

  • Use a data USB cable.
  • Close Arduino IDE.
  • Disconnect serial monitors.
  • Avoid USB hubs if possible.
  • Keep device powered during flashing.

Troubleshooting

  • Try another USB cable.
  • Install USB drivers.

  • Refresh browser and retry.
  • Verify firmware files uploaded correctly.
