Supported Hardware

Board:

ESP32-S3

Installation Steps

  • Connect the device using a USB data cable.
  • Open Chrome, Edge, Brave, or Opera.
  • Click the Connect button.
  • Select the ESP32 serial port.
  • Allow firmware installation.
  • Wait for flashing to complete.
  • The device will reboot automatically.

Browser Compatibility

  • Google Chrome
  • Microsoft Edge
  • Brave Browser
  • Opera Browser
  • Web Serial API support required

Important Notes

  • Use a USB data cable.
  • Close Arduino IDE Serial Monitor.
  • Disconnect other serial tools.
  • Do not unplug during flashing.
  • Use direct USB ports when possible.

Troubleshooting

  • Try a different USB cable.
  • Install CP210x or CH340 drivers.
  • Refresh browser and reconnect.
  • Enter bootloader mode manually.
  • Verify manifest.json loads correctly.
  • Verify BIN file URLs are accessible.
